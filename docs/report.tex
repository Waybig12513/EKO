\section{Overview}
This system uses Nicla Vision to run sensing and inference (camera + IMU + ToF) and transmits compact metadata packets to a secondary MCU (XIAO ESP32-C6) that drives haptics and communications.

\section{Dataflow}
Sensors $\rightarrow$ Vision output (class/conf/dir) $\rightarrow$ Fusion (urgency) $\rightarrow$ Packet TX $\rightarrow$ XIAO RX $\rightarrow$ Haptics.
